## 二、各能力值效果说明

| 能力值       | 效果说明                                                           |
| ------------ | ------------------------------------------------------------------ |
| 力量（力）   | 影响攻击力与命中精度                                               |
| 智慧（知恵） | 影响攻击魔法的威力                                                 |
| 精神（精神） | 影响魔法防御力与魔法回复量                                         |
| 体力（体力） | 影响最大 HP 与防御力                                               |
| 敏捷（敏捷） | 影响命中精度、回避能力、攻击次数、陷阱解除率（似乎也影响攻击顺序） |
| 运气（運）   | 影响回避率、陷阱解除率、暴击率及各类概率变化                       |

## 三、各状态加成（能力值 20 以上开始附加）

| 能力值       | 加成效果                                            |
| ------------ | --------------------------------------------------- |
| 力量（力）   | 物理：追加造成伤害                                  |
| 智慧（知恵） | 魔法：追加造成伤害                                  |
| 精神（精神） | 魔法：减少受到的伤害                                |
| 体力（体力） | 物理：减少受到的伤害；吐息（ブレス）：追加造成伤害  |
| 敏捷（敏捷） | 回避上限值（95%）提升；可回避龙息（ドラゴンブレス） |
| 运气（運）   | 不明                                                |
