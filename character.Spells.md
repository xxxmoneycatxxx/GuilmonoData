## 一、魔法师与僧侣的魔法说明（魔法使い／僧侶の魔法）

※魔法师魔法（魔法使い魔法）、僧侣魔法（僧侶魔法）在一次战斗（一回の戦闘）中，每种魔法（それぞれ一つ）各只能使用一次（一回のみ使用可能）。

## 二、魔法师的魔法（魔法使いの魔法）

| 魔法等级（魔法 Lv） | 名称（名称）                 | 说明（説明）                                                                                |
| ------------------- | ---------------------------- | ------------------------------------------------------------------------------------------- |
| Lv1                 | 魔法箭（マジックアロー）     | 发射 3 支魔法箭，对敌人造成伤害。作为基础攻击魔法，其威力受使用者（使い手）的实力影响较大。 |
| Lv2                 | 睡眠云雾（スリープクラウド） | 生成诱导睡眠的云雾，使全体敌人陷入睡眠状态。                                                |
| Lv3                 | 火球术（ファイヤーボール）   | 引发火焰爆炸，对全体敌人造成伤害。是容易习得的全体攻击魔法。                                |
| Lv4                 | 暴风雪（ブリザード）         | 引发暴风雪，对全体敌人造成伤害。                                                            |
| Lv5                 | 攻击提升（アタックアップ）   | 将队伍成员（パーティメンバー）对敌人造成的伤害提升至 2 倍。                                 |
| Lv6                 | 闪电术（サンダーボルト）     | 用强力雷击对单个敌人（敵一体）造成大量伤害。受使用者的魔法攻击力影响较大。                  |
| Lv7                 | 核能术（ニュークリア）       | 引发核爆炸，对全体敌人造成大量伤害。即使使用者的魔法攻击力较低，也能期待稳定的威力。        |

### 魔法师魔法补充说明

1. **威力排序**：火球术（ファイヤーボール）＜暴风雪（ブリザード）＜核能术（ニュークリア）

2. **魔法攻击力上升对威力的提升幅度**：核能术（ニュークリア）＜暴风雪（ブリザード）＜火球术（ファイヤーボール）

    ＝即随着魔法攻击力提高，威力排序会发生逆转。

3. 此外，魔法箭（マジックアロー）与闪电术（サンダーボルト）的威力提升效果尤为显著。

## 三、僧侣的魔法（僧侶の魔法）

| 魔法等级（魔法 Lv） | 名称（名称）               | 说明（説明）                                                                 |
| ------------------- | -------------------------- | ---------------------------------------------------------------------------- |
| Lv1                 | 治愈术（ヒール）           | 恢复一名队友（味方一人）的 HP。                                              |
| Lv2                 | 护盾屏障（シールドバリア） | 将队伍成员受到的攻击伤害（攻撃ダメージ）与吐息伤害（ブレスダメージ）减半。※1 |
| Lv3                 | 净化术（キュア）           | 治愈队伍全体（パーティ全員）的异常状态（状態異常）。                         |
| Lv4                 | 强效治愈（ヒールプラス）   | 大幅恢复一名队友的 HP。                                                      |
| Lv5                 | 魔法屏障（マジックバリア） | 将队伍成员受到的魔法伤害（魔法ダメージ）减半。※2                             |
| Lv6                 | 完全治愈（フルヒール）     | 将 HP 低于一半（半分以下）的一名队友的 HP 完全恢复。                         |
| Lv7                 | 群体治愈（パーティヒール） | 恢复队伍全体的 HP。                                                          |

### 僧侣魔法补充注释

* ※1：可与【减半】抗攻击（\[半減] 攻撃に強い）技能同时发挥效果（同時に効果を発揮します）。

* ※2：可与【减半】抗魔法（\[半減] 魔法に強い）技能同时发挥效果。
